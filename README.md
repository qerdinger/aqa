# Automated Quality Assurance (a.k.a AQA)

## How it operates

First, the project is imported and its Abstract Syntax Tree (AST) is analyzed to identify relevant methods and functions.  
Once this analysis is complete, the system determines the nature of each function and recommends appropriate testing strategies. It also generates edge-case scenarios to thoroughly evaluate them.  
Finally, a comprehensive suite of tests is automatically produced to ensure complete validation and assessment of the project.
